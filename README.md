

Projeto baseado no modelo de oficina automotiva de Thales Vaz, adaptado para **reparo de computadores**.



- Cadastro de equipamentos (notebooks, desktops, impressoras)
- Agendamento de atendimentos
- Histórico de revisões
- Consulta de status dos equipamentos


- Tabela `equipamentos`
- Tabela `agendamentos`
- Tabela `revisoes`



- `create_database.sql` — cria as tabelas
- `insert_example.sql` — insere dados de exemplo
