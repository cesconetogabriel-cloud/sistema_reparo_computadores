Sistema de Reparo de Computadores

Projeto completo com backend (Node.js + Express + PostgreSQL) e frontend (HTML, CSS, JS).

Funcionalidades
- Cadastro de equipamentos
- Registro de agendamentos
- Registro de revisões

1. Clonar o projeto:
   ```bash
   git clone https://github.com/cesconetogabriel-cloud/reparo-computadores.git

2.Backend:
cd reparo-computadores/backend
npm install
node server.js
